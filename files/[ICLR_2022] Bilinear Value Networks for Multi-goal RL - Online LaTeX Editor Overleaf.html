BILINEAR VALUE NETWORKS
Zhang-Wei Hong∗, Ge Yang* & Pulkit Agrawal
Improbable AI Lab
NSF AI Institute of AI and Fundamental Interactions (IAIFI)
Computer Science and Artificial Intelligence Laboratory (CSAIL)
Massachusetts Institute Technology
ABSTRACT
Universal value functions are a core component of off-policy multi-goal reinforce-
ment learning. The de-facto paradigm is to approximate Q(s,a,g) using mono-
lithic neural networks which lack inductive biases to produce complex interactions
between the state s and the goal g. In this work, we propose a bilinear decompo-
sition that represents the Q-value via a low-rank approximation in the form of a
dot product between two vector fields. The first vector field, f(s,a), captures
the environment’s local dynamics at the state s; whereas the second component,
φ(s,g), captures the global relationship between the current state and the goal. We
show that our bilinear decomposition scheme improves sample efficiency over the
original monolithic value approximators, and transfer better to unseen goals. We
demonstrate significant learning speed-up over a variety of tasks on a simulated
robot arm, and the challenging task of dexterous manipulation with a Shadow
hand.
1 INTRODUCTION
Learning goal conditioned policies (Kaelbling, 1993; Jordan & Rumelhart, 1992; Wolpert et al.,
1995; Schaul et al., 2015; Agrawal et al., 2016; Nair et al., 2017; Pathak et al., 2018; Andrychowicz
et al., 2017; Paster et al., 2021) is a central problem in decision making. Reinforcement learning
(RL) is a popular approach for learning goal-conditioned policies. Learning a single policy for
many goals using RL is challenging because the data requirement is substantially higher than single
task learning because the agent needs to collect data for multiple goals. To reduce the amount of
data needed for learning, it is common to leverage off-policy methods to learn a goal-conditioned Q-
value function Q(s,a,g) (or, Universal Value Function Approximator (UVFA) (Schaul et al., 2015)),
where s, a, and g denote states, actions, and goals, respectively.
To efficiently learn Q(s,a,g) for a large set of goals, it is important for the UVFA to generalize
to new goals, as suggested in Schaul et al. (2015). The number of combinations in the inputs (i.e.,
s,a and g) grows significantly than single-task learning problem since single-task problem does
not require accounting for variations of goals. The increased number of input patterns impedes the
value function learning progress. The speed of value function learning influences the progress of
policy learning, hence impacting the data collection process that is crucial for sample efficiency.
Under-performing data cannot effectively improve the agent’s performance. As a result, the ability
to generalize value predictions to unseen combinations state-actions and goals is critical for multi-
goal RL. A generalizable Q-value function can make approximately accurate value predictions to
new goals seen in the middle of training via the knowledge learned early during training.
A common approach to improve generalizability is using hindsight experience replay
(HER) (Andrychowicz et al., 2017) to train the UVFA with additional data produced by relabel-
ing, such that UVFA is exposed to more diverse combinations of goals. An alternative strategy to
achieve generalizability is disentangling the problem so that parts of the knowledge can be reused
on new instances of the problem. For example, Barreto et al. (2016) decouple the representations of
environment dynamics and rewards using successor features (Dayan, 1993), transferring the value

state.

and TD3 (Fujimoto et al., 2018) where the policy is supervised entirely by the value function on off-
policy interaction data. We show empirically that bilinear value decomposition enables the value
function to learn significantly faster than the commonly used monolithic architecture shown in Fig-
ure 1a (Section 5.1). Further, BVN also exhibits superior speed when adapting to new goals (Sec-
tion 5.2). Results are presented on two standard benchmark task suites in simulation: (i) dexterous
manipulation with a twenty degrees-of-freedom (DOF) shadow hand and (ii) object pick-place tasks
using the Fetch manipulator.

2.1 OFF-POLICY RL WITH ACTOR-CRITIC METHODS
The multi-goal continuous control tasks considered in this paper are typically solved via actor-critic
algorithms such as deep deterministic policy gradient (DDPG) (Silver et al., 2014), twin-delayed
DDPG (TD3) (Fujimoto et al., 2018) and soft actor-critic (SAC) (Haarnoja et al., 2018), to name a
few. In DDPG, the critic Q is learned by minimizing the temporal-difference (TD) loss:
L(Q) = E(st,at)∼Uniform(D)(rt,g + γQ(st+1,π(st+1,g),g) −Q(st,at,g))2, (3)
where D is the the replay buffer (Mnih et al., 2015) containing state transition tuples
(st,at,rt,st+1,g). The actor π is trained to maximize the score from the critic, using the deter-
ministic policy gradient:
∇πJ(π) = E[∇aQ(s,a,g)|a=π(s,g)]. (4)
In multi-goal RL settings, we refer to the goal-conditioned critic as the universal value function
approximator (UVFA) (Schaul et al., 2015) because Q(s,a,g) approximates the expected utility for
all goals ∈G. Because the actor relies on the critic for supervision, the ability for the critic to quickly
learn and adapt to unseen goals is a key contributing factor to the overall sample efficiency of the
learning procedure. A critic that provides better value estimates using less sample and optimization
gradient steps, will lead to faster policy improvement via the gradient in Equation 4. We refer
the reader to the original papers for the algorithmic details on SAC and TD3 because we found
empirically that DDPG works better than TD3 and SAC on these robot manipulation tasks (see
Section B).
